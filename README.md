# BetterCodePush

Over-the-air (OTA) updates for React Native — ship JavaScript and asset changes
to your users without waiting on the app stores.

- **SDK** — `@bettercodepush/react-native`: drop-in runtime that fetches and applies updates.
- **CLI** — `@bettercodepush/cli` (`bettercodepush`): build and release update bundles.

This repository is the public home for **documentation, releases, and issues**.
The published packages live on npm under the [`@bettercodepush`](https://www.npmjs.com/org/bettercodepush) scope.
